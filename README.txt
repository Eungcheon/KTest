Git 명령어

 -remote add origin 원격 저장소 주소 : 로컬저장소에 원격저장소 주소를 알려줌
   - remote : 원격 저장소를 관리하는 명령어(원격저장소란 네트워크 어딘가의 프로젝트 버전을 의미)
   - add : 새로운 원격 저장소를 추가
   - origin : 원격 저장소의 이름(다른이름을 지정할 수도 있지만 origin은 관례적으로 사용되는 기본 원격 저장소의 이름)
   - 주소 : 원격 저장소의 URL

 -push origin master : 로컬 저장소의 변경 사항을 원격 저장소로 전송
   - push : 로컬 저장소의 커밋을 원격 저장소로 전송
   - origin : 원격 저장소의 이름
   - master : push 할 로컬 브랜치의 이름(원격 저장소에 해당 이름의 브랜치가 없다면 생성됨)
   - branch 참고 링크 : https://velog.io/@euisuk-chung/Git-Branch%EC%97%90-%EB%8C%80%ED%95%9C-%EC%9D%B4%ED%95%B4%EC%99%80-%ED%99%9C%EC%9A%A9

 -clone <원격 저장소 주소> <폴더 경로>  : 폴더 경로에 원격 저장소의 전체 내용을 복사해옴
   - clone : 원격 저장소의 전체 내용을 복사
   - <폴더 경로> : 지정된 폴더가 존재하지 않으면 자동 생성, 지정하지 않으면 저장소 이름으로 폴더 생성, . 으로하면 폴더를 생성하지 않고 현재 폴더에 복사

 -pull origin master : 원격 저장소(origin)의 브랜치(master) 변경사항을 현재 로컬 브랜치로 가져와 병합(업데이트)
